# 哲思黑匣子 (Zhesi Blackbox)
基于原创理论的本地哲学思考器，纯 Python 原生运行，无需联网、无需大模型、无需 Token。

## 核心特点
- 🧠 **理论驱动，而非数据训练**：所有回答基于原创的「0×∞=1」公理体系，不是对现有哲学文本的拼接与复述。
- 🚀 **轻量本地运行**：单文件脚本，双击即可启动，无任何第三方依赖，在普通设备上秒级响应。
- 🛡️ **安全边界明确**：内置硬编码拦截，主动拒绝数学难题证明等高危场景，坚守“不打开潘多拉魔盒”的设计原则。
- 🌱 **认知深度演化**：对同一问题的多次提问，答案会随认知深度递进，从现象描述逐步逼近本质推导。

## 支持的核心哲学维度
- 我是谁？
- 人为什么存在？
- 意识的本质是什么？
- 什么是真正的自由？
- 人生的意义在哪里？

## 快速开始
1.  下载 `zhesi_blackbox.py` 文件
2.  直接运行：
    ```bash
    python zhesi_blackbox.py
    ```
3.  输入你的哲学问题，体验理论驱动的深度思考。

## 设计理念
本项目是「观测者闭合框架」的一次具象化演示。它展示了如何从一个核心公理出发，构建一套自洽、可演化的本体论思考体系，而非依赖海量数据的概率生成。

> 注：本项目为理论演示版，核心的公理落地与推导逻辑并未开源。其价值在于提供一种不同于当前大模型范式的思考路径。
